# Release Plan: v2.8.0 — Exploit Pattern RAG

## Overview
v2.8.0 introduces **Exploit Pattern RAG**, a major upgrade to the AutoPoC generation engine. This system allows SCOUT to leverage structured exploit knowledge (Patterns) to generate sophisticated, multi-stage (stateful) Proof-of-Vulnerability plugins.

## Key Features
1. **Knowledge Base (`data/exploit_references/`)**: Centralized repository for structured exploit patterns.
2. **Scoring Retriever**: Intelligently selects patterns based on candidate-target alignment (Channels, Sinks, Triggers).
3. **Adaptation Engine**: Prompts LLMs to "adapt" patterns to the current target's `Plan IR` instead of simple imitation.
4. **Contamination Guard**: Ensures reference-specific artifacts (like hardcoded IPs or endpoints) do not leak into the final PoC.
